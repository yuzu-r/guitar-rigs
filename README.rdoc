== README

MVP (improvements to come) for a pinterest clone application.
Add and remove images (via url); url is validated via FastImage. After creation, broken images are replaced with a placeholder. Authenticate via Twitter or create an account within the application (via Devise).
Built with Rails and jQuery. Image manipulation via masonry-rails.

This is part of freeCodeCamp's Backend Development curriculum. Visit at https://www.freecodecamp.com.
